Introduccion de Linux
  • ¿Que es Linux?

Linux es un sistema operativo de codigo abierto o libre, al ser un sistema operativo libre puede ser modificado o personalizado a gusto de cuaquier persona y es distribuible por cualquier persona o empresa, pero se distribuye bajo licencia de codigo abierto.

Linux es un kernel que seria el nucleo del sistema operativo que representa al menos el 50% del codigo del sistema.

  • ¿Que hace la consola o terminal de linux?

La consola o terminal en linux es una shell, que el ususario puede usar para interactuar con el ordenador usando comandos, esto se hacia hace mucho donde la los sistemas operativos usaban unterfaz de linea de comandos, pero a hora es mucho mas facil interactuar con el ordenador porque ya vienen con una interfaz grafica intuitiva.

ls y cd

Ls es unos de los comandos mas usado en la consola, este comando solo se usa para mostrar todos los archivos y carpetas dependiendo donde este ubicado el usuario, o poner ls mas la carpeta para ver los archivos que esta en esa carpeta.

user@Ubuntu~:$ls
Descargas Imágenes Música Vídeos Documentos JavaScript Plantillas yay Escritorio LuisM Público ejercicios.txt
user@Ubuntu~:$ls Descargas/
height: 100%; margin: 0 auto 0; ventoy-1.0.99 'MiniOS10 LTSC-21H2 v2024.01 x64 - www.dprojects.org.rar' www.dprojects.org.txt

Cd su usa para entrar en las carpetas, poniendo cd mas el nomber de la carpeta, no solo se usa para entrar en las carpeta tambien se usa para salir de las carpetas.

user@Ubuntu~:$cd Descargas/
user@Ubuntu/Descargas~:$ls
'MiniOS10 LTSC-21H2 v2024.01 x64 - www.dprojects.org.rar' www.dprojects.org.txt
user@Ubuntu/Descargas~:$cd
user@Ubuntu~:$ height: 100%; margin: 0 auto 0;
  • ls puede hacer otros cosa si lo acompañas con una una flag, en la ultima seccion se explicara que son las flag. La flag se usan con un comando como ls -letra o comando --palabra.
  • Se pueden usar ls -a para mostrar los archivos mas los archivos ocultos que son archivo con un punto al principio .archivo o .carpeta.
  • Cd se usa dos punto o un guion para salir de las carpetas, cd .. y cd - . Pruebenlon
cp y mv

Los comandos cp(copiar) y mv(mover) son comandos para copiar, mover y remplazar el nombre de un archivo o de una carpeta.

Estos comando se usan asi; comando archivo destino, cp o mv prueba.txt prueba.

user@Ubuntu~:$ls
prueba prueba.txt
user@Ubuntu~:$cp prueba.txt prueba
user@Ubuntu~:$ls prueba
prueba.txt user@Ubuntu~:$ls
prueba prueba.txt
user@Ubuntu~:$mv prueba.txt prueba
user@Ubuntu~:$cd prueba
user@Ubuntu/prueba~:$ls
prueba.txt
user@Ubuntu/prueba~:$mv prueba.txt pr.txt
user@Ubuntu/prueba~:$ls
pr.txt
  • Con mv se remplazo el nombre de prueba.txt a pr.txt.
  • Tambien se usas comandon secundoario para tener mas permisos y flag.
  • Como cp -rf(r(directorio y f(forzar))) archivo destino y sudo(para tener permisos) mv archivo destino o remplazo.
mkdir y touch

Estos comandos mkdir y touch se usan para crear carpetas y archivos, mkdir se usa para crear carpetas y touch para crear los archivo.

Vamos usar como ejemplo prueba y prueba.txt

user@Ubuntu~:$mkdir prueba
user@Ubuntu~:$ls
prueba
user@Ubuntu~:$cd prueba
user@Ubuntu/prueba~:$touch prueba.txt
user@Ubuntu/prueba~:$ls
prueba.txt
  • Bueno mkdir y touch no usan flag ni sudo, solo se usas para crear.
  • Se pueden crear archivos ocultos poniendole el punto al principio;
  • pureba.txt o .bashrc(archivo de configuracion).
rm y cat

El cat se usa para ver el contenido de los archivos y rm se usa para borrar archivos o carpetas.

Se uasn asi;

user@Ubuntu~:$ls
prueba.txt
user@Ubuntu~:$ cat prueba.txt
Hola Mundo user@Ubuntu~:$ls
prueba.txt
user@Ubuntu~:$ rm -r prueba.txt
  • Si el usuarios esta ubicado en el directorio que es home y users, se usa la flag -r para poder borrar un archivo del direcorio
  • Si el usuario esta en una carpeta importante del sistema operativo, se usa el sudo para darle permiso de super usuario asi poder borrar cualquier archivo.
vim y nano

Estos comandos no son comandos, con editore de texto que se ejecutan en la terminal no es lo mismo que visual studio code, nano es un editor de codigo facil de usar y ya viene instalado por defecto en linux y se usa en la terminal.

Vim a diferencia de nano, vim es un editor de texto dificil de manejer, porque solo se usan atajos de teclado, pero es un editor de texto muy rapidio y ligero para pc de gama baja. En esta epoca no se usa mucho vim, ahora se usa una mas avanzado que es neovim.

user@Ubuntu~:$nano archivo user@Ubuntu~:$nvim archivo
  • Nano te indica los atajos de teclado en la parte inferior de la termina.
  • Neovim es un editor de codigo muy personalizable, el usuario tiene la livertad de personalizarlo a su gusto.
  • Vim si es muy usado en estaopen epoca pero ahora se todo se cambiaron a neovim que es un editor de codigo avanzado.
sudo

Sudo es el comando mas usado en linux, se usa para dar permiso de superusuario asi el usuario pueda ejecutar cualquier comando en cualquier carpeta.

Tambien se puede ejecutar el usuario root poniendo;

user@Ubuntu~:$sudo comando user@Ubuntu~:$sudo su
user@Ubuntu~/home/luis:# comando
  • En root se puede ejectar cualquier comando sin usar el sudo.
  • Si perzonaliza la bash o zsh las configuraciones se tienen que hacer en root tambien para que tenga el misma diseño que el usuario.
